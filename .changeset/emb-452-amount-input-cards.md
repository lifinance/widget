Add an opt-in redesigned Send/Receive amount input card pair (`AmountInputCardPair`), enabled via `defaultUI.cardLayout` (default `false`). When disabled, the classic chain/token selectors plus the single `AmountInput` remain the default amount-entry UI. The new cards feature a token pill selector, percentage chips with gas-reserve deduction, fiat/token toggle, price (fiat) input mode, a swap button that reverses the from/to selection, and a receive card showing route price impact.
