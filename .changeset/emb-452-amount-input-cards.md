Add redesigned Send/Receive amount input cards (`AmountInputCardPair`), replacing the legacy `AmountInput`. The new card pair is the default amount-entry UI for every non-custom mode plus the custom `deposit` mode, with a token pill selector, percentage chips with gas-reserve deduction, fiat/token toggle, price (fiat) input mode, a swap button that reverses the from/to selection, and a receive card showing route price impact.
